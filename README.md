# Real-Time Object Detection

## Overview
A real-time object detection system built using Python, YOLOv8, and OpenCV.  
The project processes live video streams and detects selected objects in real time.

## Tech Stack
- Python
- YOLOv8
- OpenCV

## Features
- Real-time object detection
- Live webcam / RTSP stream support
- Object filtering
- Object counting
- FPS display

## How to Run

```bash
pip install ultralytics opencv-python
python main.py
```

## Notes

* Make sure a webcam or RTSP source is available
* Press ESC to exit the application
